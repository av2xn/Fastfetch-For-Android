# Fastfetch-For-Android
Fastfetch Magisk module compiled for Android  

How to install?  

Download the [Fastfetch For Android Magisk module for Android](https://github.com/av2xn/Fastfetch-For-Android/releases/download/v1/fastfetchforandroid.zip)  
Then install from your root manager  
It will be appear on /system/bin/fastfetch
